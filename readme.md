## About HamStation

HamStation is a modern, cross-platform amateur radio station management application featuring:

- 📻 Rig control with multi-radio support
- 📓 Logging with ADIF import/export
- 🌍 Live world map with DX cluster, POTA/SOTA, and satellite tracking
- 📡 WSJT-X and JS8Call integration
- 🏆 Achievement and awards tracking
- 👥 Multi-operator and multi-logbook support
- 🔍 Callsign lookup (FCC, HamQTH, QRZ)
